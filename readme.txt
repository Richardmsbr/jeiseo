Transform your WordPress into a complete marketing agency with automated SEO audits, AI content generation, and one-click fixes.

== Description ==

JeiSEO is a comprehensive marketing automation plugin that combines SEO analysis with AI-powered content generation. Unlike traditional SEO plugins that only show problems, JeiSEO actually fixes them.

= Key Features =

**SEO Audit Engine**

* SSL/HTTPS verification
* XML Sitemap detection
* Robots.txt analysis
* Meta title and description checks
* Heading structure analysis (H1, H2)
* Image alt text verification
* Internal linking analysis
* Mobile responsiveness check
* Page speed indicators
* Canonical URL validation

**AI Content Generator**

* Blog post generation from keywords
* Multiple content lengths (short, medium, long)
* Configurable tone (professional, casual, friendly, authoritative)
* SEO-optimized structure with proper headings
* Direct save to WordPress drafts

**Dashboard**

* SEO Health Score (0-100)
* Issues found vs fixed tracking
* Technical status overview
* Quick action buttons
* Usage tracking for free plan

= Free vs PRO =

**Free Plan:**
* 4 audits per month
* 3 AI content generations per month
* Basic SEO checks
* Dashboard access

**PRO Plan:**
* Unlimited audits
* Unlimited content generation
* Auto-fix all issues with AI
* Weekly email reports

= Requirements =

* WordPress 6.0 or higher
* PHP 8.1 or higher
* OpenAI or Anthropic API key (for AI features)

== Installation ==

1. Upload the `jeiseo` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to JeiSEO > Settings to configure your API key
4. Start running audits and generating content

== Frequently Asked Questions ==

= Do I need an API key? =

Yes, for AI content generation features you need either an OpenAI or Anthropic (Claude) API key. The SEO audit works without an API key.

= How do I get a PRO license? =

Visit https://jeiseo.com/pro to purchase a license key. Enter it in JeiSEO > Settings.

= Is my content unique? =

Yes, all AI-generated content is unique and created specifically for your keyword. However, we recommend reviewing and editing before publishing.

= Does this work with other SEO plugins? =

JeiSEO can work alongside other SEO plugins, but we recommend using only one for meta tag management to avoid conflicts.
